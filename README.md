# Hello World 🗺

Мой первый проект на GitHub — интерактивная карта мира.

## Что делает этот проект

На странице отображается карта мира. Если навести курсор на любую страну, появляется информация:

- 🏛 **Столица** страны
- 👥 **Население** в миллионах человек
- 🗣 **Основной язык**

## Какие технологии использованы

### Leaflet.js
Это библиотека для создания интерактивных карт на JavaScript. Бесплатная и с открытым исходным кодом. Именно она отвечает за отображение карты, подсветку стран при наведении и всю интерактивность.

Сайт: [leafletjs.com](https://leafletjs.com)

### OpenStreetMap
Это источник картографических тайлов (то есть картинок-кусочков карты), которые Leaflet загружает и складывает в единую карту. OpenStreetMap — бесплатный аналог Google Maps, создаваемый сообществом.

Сайт: [openstreetmap.org](https://www.openstreetmap.org)

### world-atlas + TopoJSON
Это набор данных с границами всех стран мира в формате TopoJSON. Используется для отрисовки контуров стран поверх карты.

- [world-atlas на npm](https://www.npmjs.com/package/world-atlas)
- [topojson-client на npm](https://www.npmjs.com/package/topojson-client)

## Как запустить

Просто откройте файл `index.html` в браузере — и карта заработает. Никаких дополнительных установок не нужно.
